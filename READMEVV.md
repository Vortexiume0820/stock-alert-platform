  新增三個業務層 Prometheus metrics：
  stock_price_fetch_total    # Counter，labels: stock_id / status(success|failed)
  scheduler_last_success_timestamp  # Gauge，排程成功完成的 Unix timestamp
  slack_alert_sent_total     # Counter，labels: stock_id
  - 股價查詢成功 → stock_price_fetch_total.labels(..., status="success").inc()
  - 股價查詢失敗 → stock_price_fetch_total.labels(..., status="failed").inc()
  - Slack 送出 → slack_alert_sent_total.labels(stock_id=...).inc()
  - 整輪無失敗 → scheduler_last_success_timestamp.set_to_current_time()
